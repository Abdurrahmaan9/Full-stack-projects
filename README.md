### To get started, please follow these steps:
* [x] git clone https://github.com/Full-stack-projects.
* [x] git checkout <branch-name>
* [x] cd into project folder
* [x] run npm install or yarn install (if using yarn)
* [x] npm start or yarn start (to run the application)
